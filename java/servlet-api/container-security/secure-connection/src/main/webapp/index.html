Container form authentication over secure connection

Show protected file

Before running this sample you need:
a. configure key-store; it is java platform mechanism for managing keys and certificates; Start by adding self signed certificate:
keytool -genkey -alias tomcat -keyalg RSA
This command should create .keystore file in user home directory; afterwards tomcat will look this file to find certificates.
In production you typically import certificate signed by certificate authority
b. modify tomcat server.xml configuration by removing comments around https section. Tomcat will automatically load certificate if transport quarantee is set to confidential.

<Connector port="8443" protocol="HTTP/1.1" SSLEnabled="true" maxThreads="150" scheme="https" secure="true" clientAuth="false" sslProtocol="TLS" keystorePass="qazwsx"/>
